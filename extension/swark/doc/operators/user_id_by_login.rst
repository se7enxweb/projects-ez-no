user_id_by_login
----------------

Summary
~~~~~~~
Returns the user ID of a user identified by the login name.

Usage
~~~~~
::

    user_id_by_login( login )

Parameters
~~~~~~~~~~
    =========== =============================================================== ======== =======
    Name        Description                                                     Required Default
    =========== =============================================================== ======== =======
    login       Login name of a user                                            Yes
    =========== =============================================================== ======== =======

Examples
~~~~~~~~
::

    {user_id_by_login( 'admin' )}

Returns 14.
